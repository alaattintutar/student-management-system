# Student Management System

A Java-based application to manage academic entities, people (students, staff), and handle exceptions. Uses object-oriented principles to model academic data and logic.

## Features
- **Academic Entities:** Manage courses, departments, and other academic structures.
- **People Management:** Handle student and staff information.
- **Custom Exception Handling:** Built-in specific exceptions for proper error management (`MyExceptions.java`).
- **File Parsing Integration:** Uses `FileManager` to load/save records systematically.
